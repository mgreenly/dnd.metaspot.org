# Overview

I don’t know exactly where this is leading. The idea is simply to create a deeply hyperlinked chronicle of our in-person D&D game.  My goal is to represent the game as it is, covering both the fiction of the world and the game mechanics.  Hopefully the result is something unique and interesting.

This experiment was largely inspired by [Aaron Alston’s](https://en.wikipedia.org/wiki/Aaron_Allston) [Strike Force Archive](https://highrockpress.us/products/strike-force-archives-pdf) and a few other famously [long running D&D campaigns](https://youtu.be/nJ-ehbVQYxI).

If you're new the best place to start is with the most recent [session recap](/pages/recaps.html).
