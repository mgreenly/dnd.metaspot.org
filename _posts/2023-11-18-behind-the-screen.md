# 2023-11-18 Behind The Screen

In addition to the weekly session recaps, which are intended to be player focused, I'll be doing a matching *"Behind The Screen"* post each week. These posts will be very DM centric and will focus on what ever aspects of the game I'm currently working on; rules, world building, session prep, encounter design, etc.  In some cases the post will be directly related to that weeks session in other cases it may be about future aspects of our game. Player's should probably avoid these posts.

## <a id="lazaruspit">Lazarus Pit</a>
I've been considering adding a side quest that would result in the party finding a [*Lazarus Pit*](https://dc.fandom.com/wiki/Lazarus_Pit) they could use to restore lost constitution from resurrections but now that the players have discovered the room with the multi-colored pools in the Barrowmaze I think I'll hold off on that for the time being. By the text in Barrowmaze the pool only raises the character's prime attribute but I've decided to also have it return a characters constitution to it's original value if it's been decreased for any reason. The pool can still only be use once by each character to raise their prime attribute but it can be used any number of times to restore their constitution.
